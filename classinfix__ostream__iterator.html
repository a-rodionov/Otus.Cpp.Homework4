Подробное описание

template<class T, class charT = char, class traits = std::char_traits<charT>>
class infix_ostream_iterator< T, charT, traits >

Однопроходный итератор вывода, который записывает последовательные объекты типа T в объект std::basic_ostream, используя operator<<. Дополнительный разделитель строки записывается в выходной поток перед каждой операцией записи кроме первой.
